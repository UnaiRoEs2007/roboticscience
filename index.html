Diseño web

Tengo la habilidad de poder plasmar mis pensamientos en páginas web, para mí es un reto con el que disfruto mucho.

Diseño gráfico

Puedo hacer diseños únicos en cada web. Si se me pide que haga una tarea en concreto, la puedo conseguir con facilidad.

Desarrollo

Cada web que creo la subo a internet y le pongo dominios, ya sean predeterminados o personalizados.

Códigos de programación


Los códigos que siempre uso para programar webs son:

-HTML5 y CSS

Otros que no uso tan a menudo son:

-JavaScript (incluyendo Jquery) y PHP

Hosting y dominio




Cada página web tiene un dominio propio ya sea predeterminado o de pago.
Si se quiere un dominio que sea personalizado se compra y se añade a la web.
Cada web tiene un hosting gratuito que proviene de Github Pages

Otros proyectos




Aparte de diseñar webs, hago diferentes proyectos con:
-Arduino: es una placa que puedes programar y hacer que haga lo que tú le digas.
-C++: Es un código de programación que se basa en la lógica, es el que recomiendo para empezar.
